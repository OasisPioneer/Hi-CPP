## 关于

编写这本书的初衷是为了让自己在学习 C++ 的过程中能够对于一些知识点有一个更深刻的记忆，同时也可以整理出更加系统化的 C++ 学习路线。

* 本书分为四个部分
  * **语言核心**: C++ 的基础语法和面向对象等内容
  * **网络编程**: 网络通信、异步IO、Boost.Asio
  * **数据库**: 数据存储、数据库语句……
  * **图形框架**: TUI 与 GUI
